# Magic Marquee

A magic plugin for marquee animations

## Usage

```js
import { MagicMarqueePlugin } from '@maas/vue-equipment/plugins'
import { createApp } from 'vue'

const app = createApp({})

app.use(MagicMarqueePlugin)
```

### Marquee

````html
<magic-marquee>
  <span>Content</span>
</magic-marquee>
```
````
